Glamorise using NLIDB NaLIR (Database: MAS)


This is an experimental implementation of GLAMORISE integrated with a Python port of NLIDB NaLIR using a MAS (Microsoft Academic Search) database.

You can make queries about authors, publications, journals, conferences, keywords, citations and organizations.
