Glamorise using NLIDB NaLIR (Database: ANP)


This is an experimental implementation of GLAMORISE integrated with a Python port of NLIDB NaLIR using an ANP (Agência Nacional de Petróleo - Brazilian Petroleum Agency) database.

You can make queries about oil and gas production in fields, which in turn are located in basins, which in turn are located in one or more states.

The production data is stored on a monthly basis.

A single operator has the concession for each field.
